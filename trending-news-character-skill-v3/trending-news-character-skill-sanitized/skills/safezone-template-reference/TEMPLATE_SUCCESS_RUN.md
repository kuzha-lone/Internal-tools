# Safe-Zone Workflow Reference

This reference preserves the workflow lessons from a successful run without shipping any private source materials, drafts, rendered videos, or user-specific planning files.

## Keep these rules

- create the talking-head asset first
- build the HyperFrames plan from the approved script and real duration
- treat the visible top-half viewport as `1080x960`
- keep all major top-half content inside the centered `960x890` safe zone
- use fresh HyperFrames assets and scene architecture for every run
- assemble the final `1080x1920` stacked video and then export the `1.5x` version
- QA hook, middle, and ending frames before shipping

## What was removed from this resale copy

- approved scripts from prior client runs
- planning packages tied to a private account
- rendered HyperFrames projects and final videos
- private provider IDs and account-specific setup
