# Reporting Script Framework

Use this file while running `viral-script-skill`.

## Core Formula

A strong script in this template is:

- one event
- one mechanism
- one consequence
- one advice line

Do not add a second point unless it strengthens the same mechanism.

## Drafting Sequence

1. fact kernel
2. mechanism
3. stakes
4. advice mode
5. opener choice
6. final script

## Opener Selection Guide

Use `The system...` when the narrative is primarily explanatory.

Use `The system is testing you today...` when the narrative implies:

- pressure
- discipline
- sacrifice
- a near-term decision

Use `There was a glitch in the system today...` when the narrative exposes:

- contradiction
- narrative manipulation
- obvious absurdity
- a break between official narrative and underlying reality

## Story Archetypes

### Automation Pressure

Use when:

- a new model, agent, or workflow changes labor value

Typical emphasis:

- workers are being repriced
- tool fluency is no longer optional

Typical advice mode:

- preparatory or defensive

### Market Trap

Use when:

- hype, volatility, or public excitement is outrunning the structure

Typical emphasis:

- emotion is the trap
- timing and discipline matter more than excitement

Typical advice mode:

- skeptical or defensive

### Policy Or Macro Pressure

Use when:

- rates, inflation, jobs, debt, or regulation are shifting the field

Typical emphasis:

- the audience is paying for the change whether they notice or not

Typical advice mode:

- defensive

### Platform Or Infrastructure Shift

Use when:

- access, distribution, compute, or tool infrastructure is changing

Typical emphasis:

- leverage is concentrating
- dependence is becoming more dangerous

Typical advice mode:

- preparatory, skeptical, or opportunistic depending on the signal

## Advice Fit Test

The advice close should answer:

- what should the audience protect?
- what should the audience build?
- what should the audience delay or avoid?

If none of those are clear, the script is underdeveloped.

## Delivery Notes

Default delivery:

- low smile
- high control
- no hype cadence
- land the last sentence like instruction, not inspiration

Increase intensity only when the narrative clearly justifies it.
