# Start Here

1. Copy `.env.example` to `.env`.
2. Fill only the provider keys selected for this pipeline.
3. Edit `config/*.json`.
4. Run `python3 scripts/setup_check.py`.
5. Tell the agent to read `AGENT_RUNBOOK.md`, `SYSTEM.md`, and the central skill file.

The final video export for this pipeline is always `.mp4`.
