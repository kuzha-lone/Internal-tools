# Optional HyperFrames Animation

HyperFrames is optional. Use it only when the user wants custom animation or motion design.

## Good Fits

- animated explainers
- dashboards
- motion graphics
- title cards
- kinetic text
- editorial overlays
- visual scene beats synced to narration

## Poor Fits

- simple slideshow
- raw generated clip sequencing
- podcast clip edit
- screen recording cleanup
- basic montage

## Pipeline Rules

If selected, create an animation subskill that defines:

- input: script, beat cards, or timeline
- output: rendered visual clip or sequence
- frame size and FPS
- timing source
- validation command
- handoff path to FFmpeg export

Use HyperFrames docs and CLI when authoring/rendering actual compositions.
